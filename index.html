User Auth Start Kit

User auth can be intimidating for those who haven't done it before and cumbersome for those who don't do it often, even very basic anonymous auth. This starter kit aims to speed up the process of building a fun project that could benefit from the boilerplate functionality of user authorization. Firebase has a lot of auth options, so this repo could easily be repurposed to use google, github, email authorization, or any combination (plus many others).
